# EcomWiki — Personal Ecommerce Knowledge Base

A personal ecommerce knowledge base covering China / Korea / Japan / North America Amazon markets.

## 🚀 Quick Start (GitHub Pages)

1. Fork or upload this repo to your GitHub account
2. Go to **Settings → Pages → Source → main branch / root**
3. Your Wiki is live at: `https://[username].github.io/ecomwiki/`

## 📖 Using with the Skill

1. Install the EcomWiki Skill in Claude.ai Projects (see `ecomwiki-skill.md`)
2. Ask any ecommerce term: "什么是 ROAS？"
3. Copy Claude's full response
4. Open your EcomWiki URL → click **「从剪贴板导入 AI 回复」**
5. Entry is imported automatically ✅

## Features

- **API Key Mode**: Direct query via Anthropic / OpenRouter API
- **Manual Paste Mode**: Copy prompt → paste to any AI → paste response back
- **Clipboard Import**: One-click import after copying AI response
- **Enterprise SSO Mode**: Works with Claude Enterprise (no API key needed)
- **4-Market Analysis**: CN / KR / JP / US per entry
- **Local Storage**: All data stored in browser, no server needed
- **Export**: Single entry as HTML, all entries as JSON backup

## Data & Privacy

All entry data is stored in your browser's `localStorage`. Nothing is sent to any server except direct API calls (which go to the AI provider you configure). Your API key is also stored locally only.
